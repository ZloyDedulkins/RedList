# RedList — отчёт по сотрудникам

Статическая страница, которая:
- читает данные из Google Sheets;
- находит максимальную дату в основном листе;
- показывает **две вкладки**:
  - «Без обратной связи»: записи на максимальную дату, у которых пустые поля «Причина» и «Статус»;
  - «Были ранее в других списках»: записи на максимальную дату по сотрудникам, которые уже встречались в более ранних выгрузках;
- выводит таблицы с нужными полями.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8080
```

Для выкладки можно использовать готовый фронтенд из `dist/` (файлы `index.html`, `app.js`, `styles.css`).

## Настройка

В `app.js` проверьте значения:
- `CONFIG.mainSheetName` — имя основного листа (по умолчанию `Список`)
- `CONFIG.mainSheetGid` — если указать `gid`, загрузка пойдёт по `gid` (это надёжнее, чем по имени листа)
- `CONFIG.mainColumns` — названия ключевых столбцов основного листа (включая `Состояние`, если колонка присутствует)

Если ваш основной лист называется иначе, поменяйте `mainSheetName`.
Если по имени лист всё равно не находится, укажите `gid` вкладки (число из URL Google Sheets после `gid=`).
