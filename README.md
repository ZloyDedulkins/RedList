# RedList — первая страница

Статическая страница, которая:
- читает данные из Google Sheets;
- находит максимальную дату в основном листе;
- фильтрует сотрудников с пустым комментарием на эту дату;
- подтягивает нормализованное подразделение с листа **«Мост (Имена подразделений)»**;
- показывает результат в формате `ФИО - Подразделение`.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8080
```

## Настройка

В `app.js` проверьте значения:
- `CONFIG.mainSheetName` — имя основного листа (по умолчанию `Список`)
- `CONFIG.bridgeSheetName` — имя моста (`Мост (Имена подразделений)`)

Если ваш основной лист называется иначе, поменяйте `mainSheetName`.
